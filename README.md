# 🏥 Healthcare Analytics Dashboard

## 📌 Project Overview
An interactive Power BI dashboard developed to analyze healthcare data, patient demographics, hospital efficiency, clinical performance, and patient readmission risks.

## 🛠 Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel / CSV
- Data Visualization

## 📊 Dashboard Pages

### 1. Clinical and Operational Performance
Analyzes KPIs related to hospital operations, treatment outcomes, and clinical performance.

### 2. Hospital Efficiency and Analysis
Provides insights into hospital resource utilization, efficiency metrics, and operational trends.

### 3. Patient Demographics and Satisfaction
Examines patient age groups, gender distribution, satisfaction scores, and demographic patterns.

### 4. Prediction Readmission and Risk
Identifies high-risk patients and analyzes factors influencing readmission.

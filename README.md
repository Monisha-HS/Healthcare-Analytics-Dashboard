# 🏥 Healthcare Analytics Dashboard

Interactive Power BI dashboard developed to analyze healthcare data, patient demographics, hospital efficiency, clinical performance, and patient readmission risks.

---

## 📌 Project Overview

This project provides comprehensive insights into:

- Clinical and operational performance
- Hospital efficiency analysis
- Patient demographics and satisfaction
- Prediction of patient readmission risks

The dashboard enables healthcare organizations to monitor key performance indicators and make data-driven decisions.

---

## 🛠 Tools & Technologies Used

- Power BI
- DAX
- Power Query
- Excel / CSV
- Data Visualization
